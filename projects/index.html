Projects

Here are some things I've (publically) worked on over the years:

  • NATSrun - A lightweight TypeScript library that provides Express/Koa-like routing capabilities for NATS messages using pattern matching.
  • NimAWS - A Nim client library for AWS, specifically S3, that was built to upload images from raspberry pi camera kiosks.
  • Hexadecimal Diceware - Hexadecimal diceware word list/book and passphrase generator.
  • Occam-RA-D3 - A simple D3 pairwise data model explorer based on Occam DMM and Reconstructability Analysis.
